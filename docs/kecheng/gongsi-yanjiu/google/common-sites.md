### 一、AI 模型与生成式平台

| 平台名称                        | 链接                                                                                 | 主要用途                                        |
| --------------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------- |
| **Gemini**                  | [https://gemini.google.com](https://gemini.google.com)                             | Google 主力多模态大模型（文本、代码、图像、音频、视频）             |
| **Google AI Studio**        | [https://aistudio.google.com](https://aistudio.google.com)                         | Gemini API 的实验与开发环境，可直接生成代码、测试对话、导出 API key |
| **Vertex AI**               | [https://cloud.google.com/vertex-ai](https://cloud.google.com/vertex-ai)           | 企业级 AI 平台，支持训练、部署、调优、管控生成式模型                |
| **Kaggle**                  | [https://www.kaggle.com](https://www.kaggle.com)                                   | 数据科学社区与竞赛平台，提供免费 GPU、数据集与 Notebook            |
| **Colab（Colaboratory）**     | [https://colab.research.google.com](https://colab.research.google.com)             | 在线 Jupyter Notebook，支持 Python + GPU/TPU 实验  |
| **TensorFlow**              | [https://www.tensorflow.org](https://www.tensorflow.org)                           | Google 开源机器学习框架，深度学习与分布式训练核心库               |
| **JAX**                     | [https://github.com/google/jax](https://github.com/google/jax)                     | 高性能自动微分与加速库，用于训练大模型和强化学习                    |
| **MediaPipe**               | [https://developers.google.com/mediapipe](https://developers.google.com/mediapipe) | 跨平台视觉、语音、姿态等 ML 管线框架                        |
| **TFLite（TensorFlow Lite）** | [https://www.tensorflow.org/lite](https://www.tensorflow.org/lite)                 | 移动端/边缘设备模型部署与量化工具                           |
