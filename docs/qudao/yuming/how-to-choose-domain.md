## 二、热门顶级域名（TLD）的推荐与用途

### 1) 通用型（品牌首选）

- **.com** — 全球最强认知、SEO友好
- **.net / .org** — 通用备用，尤其组织机构
- **.co** — 初创/品牌更易获取短域名

### 2) 垂直/行业型（突出定位）

- **.ai** — AI、科技、产品演示
- **.io** — 开发者/平台型产品
- **.app** — App推介页、移动优先
- **.tech / .design / .dev** — 明确领域标签

### 3) 区域/本地化域名

- **.cn / .com.cn / .中国** — 国内品牌与SEO优先
- **.us / .uk / .de** — 本地市场定位

------

## 三、域名选购 Pragmatic 策略

### 1) 越短越好

- 3–12 字母为最佳平衡
- 纯字母优先，避免连字符/数字（除非有品牌含义）

### 2) 品牌可识别

- 易拼写、无歧义
- 国际化（拼写在不同语言不会产生负面意思）

### 3) 商标与法律风险筛查

- 在主要市场检查商标冲突
- 避免侵权和潜在诉讼成本

### 4) 组合与备用域

- 主域 + 相关 TLD（如 .com + .net + .ai）保护品牌
- 保留 misspell 版本防止流量劫持

## 五、价格与续费风险（要提前规划）

- **.com** 通常首年 ~$8–13，续费 ~$12–18
- **.io / .ai** 首年 ~$30–70，续费更高（注意未来成本）
- **隐私保护** 有的免费，有的 ~$2–$8/年
- **促销坑**：首年低价，续费大涨 → 建议选有透明续费政策的注册商

------

## 六、部署与后续准备

1. **DNS 设定**
   - 设定 A/AAAA、CNAME、MX（如果有邮箱）
   - 启用 Cloudflare Anycast DNS 提升全球访问
2. **SSL/TLS**
   - Let’s Encrypt 免费证书
   - 或用 Cloudflare 自动管理
3. **自动续费 + 绑定支付**
   - 防止丢失关键域名
   - 设置到期提醒（T-30/7/1 天）
4. **whois 隐私**
   - 保护隐私并减少垃圾邮件

------
